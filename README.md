# ferret-cli-hello-world

## Description
A POC for ferret programming language.

## Tech stack
- g++
- cpp
- java

## Docker stack
- ubuntu:22.04
- rzuckerm/ferret

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
